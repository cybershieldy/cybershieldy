# 🚦💻 Şəhər Sistemləri Hack Oluna Bilərmi?

Bugünkü dünyada artıq şəhərlər **smart city** sistemləri ilə idarə olunur:

- 🚦 işıqforlar  
- 📹 müşahidə kameraları  
- 📺 böyük reklam ekranları  
- 📡 IoT cihazlar  

Bəs sual:

> ❗ Bunlar hack oluna bilərmi?

## 🔥 Cavab:
➡️ **BƏLİ — amma bu düşündüyün qədər sadə deyil**

---

# 🧠 Smart City Sistemləri Necə İşləyir?

Smart şəhər sistemləri əsasən aşağı komponentlərdən ibarətdir:

- IoT cihazlar (sensorlar)  
- Mərkəzi idarəetmə serverləri  
- Şəbəkə infrastrukturu  
- Web və ya SCADA panellər  

💡 Yəni bu sistemlər = **şəbəkə + proqram + hardware**

---

# ⚠️ Potensial Hücum Nöqtələri

## 🔴 1. Şəbəkə Boşluqları

Əgər sistemdə:

- açıq portlar varsa  
- firewall zəifdirsə  
- default konfiqurasiya qalırsa  

➡️ hacker giriş əldə edə bilər

---

## 🔴 2. IoT Zəiflikləri

IoT cihazların problemi:

- default şifrələr  
- update olunmur  
- zəif firmware  

💥 Bu = ən zəif nöqtə

---

## 🔴 3. Web Panel Hücumları

- bruteforce  
- SQL injection  
- auth bypass  

➡️ idarəetmə paneli ələ keçirilə bilər

---

## 🔴 4. Fiziki Hücumlar

- Bad USB  
- cihazlara birbaşa giriş  

💥 Bu ən təhlükəli ssenarilərdən biridir

---

# 🚦 İşıqforlar Hack Oluna Bilərmi?

✔️ Teorik olaraq mümkündür

Necə?

- SCADA sistemlərinə giriş  
- PLC exploit  
- şəbəkə manipulation  

💥 Nəticə:

- trafik qarışır  
- qəzalar baş verə bilər  

---

# 📺 Böyük Reklam Ekranları

Ən çox hack olunan sistemlərdən biri 😄

Necə?

- açıq admin panel  
- zəif login sistemi  
- remote access  

💥 Nəticə:

- ekran dəyişir  
- mesajlar dəyişdirilir  

---

# 📹 Şəhər Kameraları

Ən kritik hissə 🔥

Hack edilərsə:

- izləmə sistemi deaktiv olur  
- məlumatlar oğurlanır  
- real-time manipulyasiya edilir  

---

# 🌍 Bu Hücumlar Qlobal Ola Bilərmi?

✔️ BƏLİ, əgər:

- sistem internetə açıqdır  
- eyni vendor istifadə olunur  
- patch yoxdur  

💡 Bir exploit → minlərlə cihaz

---

# 🧬 Niyə Hər Kəs Edə Bilmir?

Çünki:

- sistemlər çox vaxt daxili şəbəkədədir  
- ciddi firewall var  
- dövlət səviyyəsində qorunur  

---

# 🛡️ Necə Qorunurlar?

## 🔵 Network Segmentation
Sistemlər internetdən ayrılır  

## 🔵 Firewall + IDS/IPS
trafik izlənir  

## 🔵 SIEM (Wazuh və s.)
anomaliya detection  

## 🔵 Access Control
yalnız icazəli giriş  

## 🔵 Monitoring
24/7 nəzarət  

---

2. Port scanning  
3. Service analysis  
4. Exploit axtarışı  
5. Access əldə etmə
