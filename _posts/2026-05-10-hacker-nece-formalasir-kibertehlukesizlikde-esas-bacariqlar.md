# 💀 Hacker Necə Formalaşır?

Bir çox insan düşünür ki:

❌ Hacker olmaq = proqram açmaq  
❌ Bir düymə ilə sistemi sındırmaq  
❌ Hazır tool istifadə etmək

Amma reallıq tam fərqlidir.

Əsl kibertəhlükəsizlik mütəxəssisi:

🧠 düşünür  
📡 analiz edir  
⚡ sistemləri başa düşür  
🛡️ müdafiəni öyrənir  
💻 laboratoriya qurur  
🔍 log analiz edir  

Yəni hacker olmaq:
sadəcə “hack etmək” deyil…

➡️ texnologiyanı dərindən anlamaqdır.

---

# 🛡️ Kibertəhlükəsizlikdə Ən Vacib Bacarıqlar

## 🌐 1. Şəbəkə Bilikləri (Networking)

Əgər şəbəkəni bilmirsənsə:

➡️ hücumun necə işlədiyini də anlaya bilməzsən.

Öyrənilməli mövzular:

- IP Addressing
- TCP / UDP
- DNS
- HTTP / HTTPS
- VLAN
- Routing
- Switching
- Firewall

⚡ Hackerlər ilk olaraq trafiki analiz etməyi öyrənirlər.

---

# 🐧 2. Linux Bilikləri

Kibertəhlükəsizlik dünyasının əsas sistemi:

🔥 Linux

Çünki:

- serverlərin çoxu Linuxdur
- təhlükəsizlik alətləri Linuxda işləyir
- log analizləri Linux üzərində edilir

Öyrənilməli mövzular:

```bash
cd
ls
grep
chmod
systemctl
journalctl
ssh
tcpdump
```

➡️ Linux bilmədən professional cyber security inkişafı çox çətindir.

# 🔴 Red Team Nədir?

Red Team:

➡️ hücum tərəfidir.

Məqsəd:

- sistemdə zəiflik tapmaq
- real hücum simulyasiyası etmək
- şirkətin müdafiəsini test etmək

İstifadə olunan sahələr:

- penetration testing
- phishing simulation
- Active Directory attacks
- lateral movement
- privilege escalation

---

# 🔵 Blue Team Nədir?

Blue Team:

🛡️ müdafiə tərəfidir.

Onların işi:

- log izləmək
- SIEM analiz etmək
- alertlərə baxmaq
- hücumu aşkarlamaq
- sistemi qorumaq

İstifadə olunan texnologiyalar:

- Wazuh
- Splunk
- ELK Stack
- Suricata
- Zeek
- Sysmon

---

# 📡 IoT Təhlükəsizliyi Niyə Vacibdir?

Gələcək:

⚡ IoT + AI + Smart Systems

deməkdir.

IoT cihazları:

- kameralar
- sensorlar
- smart evlər
- RFID sistemləri
- ESP32 layihələri

hamısı potensial riskdir.

---

# 🤖 IoT Security üçün Nələr Lazımdır?

- Embedded systems anlayışı
- ESP32 / Raspberry Pi bilikləri
- MQTT
- HTTP API
- Sensor logic
- Network monitoring
